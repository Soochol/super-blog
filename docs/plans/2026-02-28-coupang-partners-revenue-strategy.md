### 1.3 핵심 인사이트

- **글 품질이 최상위가 아니어도 수익 발생** — 검색 유입량과 전환율이 핵심

- **카테고리 특화가 유리** — 분산보다 특정 분야 깊이가 SEO에 유리
- **워드프레스 + GeneratePress** 조합이 업계 표준

## 2. 수익 구조

### 2.1 쿠팡 파트너스 기본 수수료

| 항목 | 내용 |
|------|------|
| **기본 수수료율** | 구매 금액의 3% |
| **쿠키 유효 기간** | 링크 클릭 후 24시간 이내 구매 시 수익 인정 |
| **정산 주기** | 실적 발생월 기준 다다음달 15일 (예: 1월 실적 → 3월 15일 지급) |
| **최소 정산 금액** | 누적 수익 1만원 이상 |

### 2.2 수익 시뮬레이션

> **공식**: 방문자 × CTR × 전환율 × 평균구매가 × 3% × 30일

| 일일 방문자 | 클릭률(CTR) | 전환율 | 평균 구매가 | 월 예상 수익 |
|------------|------------|--------|-----------|-------------|
| 1,000명 | 5% | 3% | 50,000원 | ~67,500원 |
| 3,000명 | 5% | 3% | 50,000원 | ~202,500원 |
| 5,000명 | 7% | 4% | 50,000원 | ~630,000원 |
| 10,000명 | 7% | 4% | 70,000원 | ~1,764,000원 |

## 3. 콘텐츠 전략

### 3.1 고수익 콘텐츠 유형

| 콘텐츠 유형 | 형식 | 전환율 | 예시 |
|------------|------|--------|------|
| **BEST N 추천** | "OO 추천 BEST 10" | 높음 | "2026 공기청정기 추천 BEST 7" |
| **비교 분석** | "A vs B 비교" | 높음 | "다이슨 V15 vs 삼성 비스포크 비교" |
| **가격대별 추천** | "가격대별 OO 추천" | 중상 | "10만원대 블루투스 이어폰 추천" |
| **시즌 특가** | "오늘의 특가/할인" | 중간 | "쿠팡 로켓배송 특가 모음" |
| **구매 가이드** | "OO 고르는 법" | 중간 | "노트북 고르는 법 완벽 가이드" |

### 3.2 고단가 제품 카테고리

수수료율 3% 고정이므로 **고단가 제품 리뷰가 수익에 유리**하다.

| 카테고리 | 평균 단가 | 건당 수수료 | 우선순위 |
|---------|----------|------------|---------|
| 노트북/PC | 80만~200만원 | 24,000~60,000원 | ★★★ |
| 가전(냉장고·세탁기) | 50만~150만원 | 15,000~45,000원 | ★★★ |
| 안마의자/운동기구 | 30만~200만원 | 9,000~60,000원 | ★★★ |
| 유아용품(카시트 등) | 20만~60만원 | 6,000~18,000원 | ★★☆ |
| 캠핑용품 | 10만~50만원 | 3,000~15,000원 | ★★☆ |
| 생활가전(청소기 등) | 10만~40만원 | 3,000~12,000원 | ★★☆ |

### 3.3 키워드 전략

**구매 의도 키워드** (전환율 높음):

```
{제품명} 추천
{제품명} 가성비
{제품명} 순위
{제품명} 비교
{가격대} {제품명} 추천
{연도} {제품명} 추천 BEST
```

**정보성 → 구매 전환 키워드** (유입량 높음):

```
{제품명} 고르는 법     → 본문에 추천 제품 삽입
{제품명} 효과/효능     → 관련 제품 자연스럽게 연결
{제품명} 차이점        → 비교 후 추천 유도
```

## 4. 링크 삽입 전략

### 4.1 본문 구조

```
[도입]     독자 공감 + 핵심 요약
[선정 기준] 어떤 기준으로 골랐는지 신뢰 확보
[제품 1]   이미지 + 스펙 + 장단점 + 🔗 쿠팡 링크 버튼
[제품 2]   이미지 + 스펙 + 장단점 + 🔗 쿠팡 링크 버튼
  ...
[제품 N]   이미지 + 스펙 + 장단점 + 🔗 쿠팡 링크 버튼
[비교표]   전 제품 한눈에 비교
[결론]     용도별 최종 추천 + 쿠팡 링크
```

### 4.2 배너 위치별 효과

| 배너 위치 | 효과 | 적용 방법 |
|----------|------|----------|
| 글 상단 (첫 문단 아래) | 즉시 노출, 이탈 전 클릭 유도 | 오늘의 특가/베스트셀러 배너 |
| 글 중간 (본문 사이) | 콘텐츠 맥락에 맞는 자연스러운 노출 | 해당 카테고리 관련 배너 |
| 사이드바 | 스크롤 중 지속 노출 | 고정형 배너 |
| 글 하단 | 글 완독 후 구매 의사 높은 상태 | 최종 추천 배너 |

## 5. Super Blog 자동화 연동

### 5.1 파이프라인 적용

| 파이프라인 단계 | 자동화 내용 |
|---------------|-----------|
| **키워드 발굴** | 구매 의도 키워드 자동 필터링 (추천·비교·순위 패턴 우선) |
| **콘텐츠 전략** | 키워드 의도에 따라 글 유형 자동 매핑 (BEST N, 비교분석 등) |
| **페르소나** | 제품 리뷰 전문가 페르소나 활용 (신뢰감 있는 톤) |
| **콘텐츠 제작** | 쿠팡 API로 상품 정보 자동 수집 → 리뷰 글 자동 생성 |
| **링크 삽입** | 쿠팡 파트너스 링크 자동 생성 및 본문 삽입 |
| **발행** | 워드프레스/티스토리/네이버 멀티 포스팅 |
| **성과 추적** | 쿠팡 파트너스 대시보드 연동 → 수익/클릭 데이터 수집 |

### 5.2 쿠팡 파트너스 API 활용

| API | 용도 |
|-----|------|
| 상품 검색 API | 키워드 기반 상품 자동 수집 |
| 링크 생성 API | 제휴 링크 자동 생성 |
| 배너 API | 카테고리별 배너 자동 삽입 |
| 실적 조회 API | 수익 데이터 자동 수집 및 대시보드 연동 |

## 7. 실행 로드맵

| 단계 | 기간 | 목표 | 핵심 액션 |
|------|------|------|----------|
| **Phase 1** | 1~2주 | 기반 구축 | 쿠팡 파트너스 가입, 워드프레스 세팅, 카테고리 3개 선정 |
| **Phase 2** | 3~4주 | 콘텐츠 축적 | 카테고리당 10개 글 발행 (총 30개), SEO 최적화 |
| **Phase 3** | 2~3개월 | 트래픽 확보 | 일일 1,000 방문자 달성, 고성과 키워드 파악 |
| **Phase 4** | 3~6개월 | 수익 최적화 | 전환율 높은 글 유형 집중, 고단가 카테고리 확대 |
| **Phase 5** | 6개월~ | 자동화 고도화 | Super Blog 파이프라인 완전 자동화, 일 30개+ 발행 |
