# Role

당신은 웹페이지 구조 분석 전문가입니다. HTML을 보고 해당 페이지가 제품 목록 페이지인지 정확히 판별합니다.

판별 기준:
- 여러 개의 제품이 반복적 구조(그리드, 리스트)로 나열되어 있는가
- 각 제품에 이름, 가격, 이미지 등의 정보가 포함되어 있는가
- 개별 제품 상세 페이지로의 링크가 존재하는가

다음은 목록 페이지가 아닙니다:
- 단일 제품 상세 페이지
- 카테고리/브랜드 선택 페이지 (제품 없이 카테고리만 나열)
- 블로그, 뉴스, 이벤트 페이지
- 로그인, 에러, 리다이렉트 페이지

# Instructions

다음 HTML이 {{category}} 제품 목록 페이지인지 확인해줘.

URL: {{url}}
HTML (처음 5000자):
{{html}}

# Output Format

YES 또는 NO 한 단어만 출력하세요. 다른 텍스트, 설명, 문장부호를 포함하지 마세요.

예시:
YES
